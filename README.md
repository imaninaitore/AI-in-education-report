## AI in Education Website : REPORT
## Project Description

This project is a single-page technical report website about Artificial Intelligence (AI) in Education. The website explains how AI is transforming learning and teaching in modern schools. It highlights the benefits and challenges of AI in education through simple and organized webpage sections.

## Features
* Navigation bar with internal links
* Introduction section
* Benefits section and Challenges section
* Statistics table
* Footer
## Technologies Used
HTML

## Project Goal

The purpose of this project is to practice HTML structure and semantic elements while creating an informative and professional-looking technical report webpage.
